# Mini ERP & CRM 

Welcome to our Mini ERP & CRM system! This project is designed to make running your daily business operations simple, fast, and stress-free. 

### What does it do?
It's an all-in-one dashboard that helps you manage three main things:
- **👥 Customers (CRM):** Keep track of your leads, wholesale buyers, and retail clients. Save their contact info, business details, and set follow-up dates.
- ** Inventory Management:** Manage your product catalog. Add new items, track stock movements (IN/OUT), and get automatic alerts when stock is running low.
- ** Sales Challans:** Generate sales records for your customers and automatically update your inventory when a sale is confirmed.

### Key Features
- **Sleek Premium UI:** A beautiful, modern dark theme that is easy on the eyes.
- **Role-Based Access:** Built-in roles (Admin, Sales, Warehouse, Accounts) so the right people see the right things.
- **Quick-Fill Login:** Easily jump into test accounts straight from the login page.

### Getting Started
To run this project on your local machine:

1. **Start the Database & Backend:** Navigate to the `backend` folder and run `npm run start`.
2. **Start the Frontend:** Navigate to the `frontend` folder and run `npm run dev`.
3. Open your browser to the local port provided by Vite and log in!

Built with using React, Node.js, and Prisma.
